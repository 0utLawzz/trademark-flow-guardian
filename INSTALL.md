# Installation Guide

## Requirements

- Node.js 18+ or Bun
- Git
- Supabase account (or local stack)

## Setup

```bash
git clone https://github.com/0utLawzz/trademark-flow-guardian.git
cd trademark-flow-guardian

# Install using bun (recommended)
bun install
```

## Configure Environment

Copy the `.env` template and add your Supabase URL and Anon Key.

## Run

```bash
npm run dev
```
The app will be available at `http://localhost:5173`.
