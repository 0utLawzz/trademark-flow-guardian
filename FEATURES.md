# Features

- Modern React UI built with Vite and Tailwind CSS.
- Client-Group Dashboard for managing trademark workflows.
- Case assignment functionality with modal support.
- Real-time Supabase backend integration.
- Advanced filtering and search capabilities (Journal No, TM No).
- Print and PDF export features with professional layout (brand logo, contact details).
- Git pre/post-commit hooks for automated GitHub pushing.
